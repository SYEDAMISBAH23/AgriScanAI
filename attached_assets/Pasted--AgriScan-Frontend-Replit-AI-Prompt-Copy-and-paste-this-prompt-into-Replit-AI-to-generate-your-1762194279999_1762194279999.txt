## BACKEND ENDPOINTS AVAILABLE:

**POST /infer_image**
- Send: image file + optional manual_plu
- Returns: produce_label, organic_prediction, detected_plu, verdict, recommendation

**POST /verify_plu**
- Send: plu_code
- Returns: plu_meaning, is_organic, explanation

**POST /login**
- Send: email, password
- Returns: success, user info

**GET /health**
- Returns: server status
